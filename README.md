### LLMAgent（名称暂定） 
计划开发基于LLM的Agent  
1. 由于多模态大模型还难以在特定的视觉场景落地，而特定场景在短时间又缺乏足够的数据训练相关多模态大模型。  
2. 因此本仓库计划复现React范式，结合LLM的planning能力，探索其在特定行觉场景的应用。  
### 开发过程
计划基于langchain实现一个简单的agent，给LLM加入视觉的能力。  

**已完成**：  
- 一些视觉工具的demo。
- 基于ollama调用本地大模型
